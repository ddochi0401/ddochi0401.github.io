Google search console 설정

구글에서 검색된 데이터를 볼 수 있는 Google Search Console에 sitemap을 등록했다.

내가 사용하고 있는 clean blog 테마에서는 jekyll-sitemap plugin이 설치되어 있기때문에 plugin 설치방법은 생략했다.

1.

_config.yml 파일에 위와 같이 설치가 되어있다.

2.

Google Search Console에 접속 후 시작하기

3.

URL 접두어 입력란에 위와 같이 자신의 Github blog 주소 입력 (ex. https://yourname.github.io/)

4.

빨간색 박스를 클릭(html파일이 다운로드가 된다.)

5.

다운로드 받아진 html파일을 자신의 Github blog 최상위 디렉토리에 올린 후 확인

6.

좌측 사이드 메뉴에 Sitemaps 클릭

7.

새 사이트맵 추가에 sitemap 입력 후 제출

8.

바로 아래에 제출된 사이트맵으로 성공된 것을 볼 수 있다.

9.

사이트맵을 제출은 했지만 바로 검색은 되지 않는다. (시간이 필요해보인다.)

추후에 google에 site:본인 도메인을 검색해본다. (ex. site:yourname.github.io)
